Zero-build static starter

Solar Maps

A lightweight static landing page for Solar Maps. Drop in your API key, share onboarding links, and ship without touching npm installs or build tooling.

API key not set
Mode fully static (HTML + CSS + JS)

Set your key once

Open config.js and replace apiKey with your provider key. The page reads that value at load time so you can confirm it is wired correctly.

Works offline

No dependencies and no package manager required. Double-click index.html or serve the folder with your favorite static server.

Share quick links

Provide teammates with documentation links, support contacts, and any other resources needed to explore solar potential data.

Optional container

A tiny nginx-based Dockerfile is included so you can deploy this static site anywhere without invoking a JavaScript toolchain.

Getting started

  1. Edit config.js to store your API key.
  2. Open index.html in a browser (no build step required).
  3. Share the page with teammates or deploy using the provided Dockerfile.
